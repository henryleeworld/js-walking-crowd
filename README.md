# JS 行走群眾

走路是世界上最佳運動之一，簡單易行，且不受時間和地點的限制，不論男女老少，什麼時候開始這項運動都不晚。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
